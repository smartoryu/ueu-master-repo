### Pelajari Modul, berikan contoh program perulangan secara mnyeluruh

#### code

```py
def perulangan():
    for i in range(0, 5):
        print("Perulangan ke-%d" % (i+1))
    else:
        print("Perulangan selesai")

perulangan()
```

[code](code.py)

#### output:

```
Perulangan ke-1
Perulangan ke-2
Perulangan ke-3
Perulangan ke-4
Perulangan ke-5
Perulangan selesai
```
